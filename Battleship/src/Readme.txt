How to Compile and Run:
    To run, run the Game class. Javac Game.java, Java Game.java

Assumptions:
    Assume that the user inputs correctly. For example, it will ask for the letter d or f,
    the user will only put in d or f. Assume user stays in range when firing.

Known Bugs or Defects:
    - Scanner does not tell the length of the boat
    - Placeboats works as intended but doesn't place vertical boats
    - There is one bug where missile goes out of range on the upper left of the board
